
Описание параметров в файле конфигурации cfg.ini

Базовая настройка для взаимодействия с Телеграмм - https://youtu.be/uoVNGwpeehI

1. Токен вашего Телеграмм-Бота - "Token","0"
2. ID чата для получения оперативного оповещения - "Chat_ID","0"
3. ID чата для получения информации о статусах - "Group_ID","0"

4. Количество установленных видеокарт в системе - "GpuInstall","1"
5. Дневной тариф за киловат в копейках - "Pay_Day","341"
6. Ночной тариф за киловат в копейках - "Pay_Night","197"
7. Как выводить на экран затраты за электроэнергию, валюта в месяц - "money_mode","Rub/m"
8. Имя Программы разгона без exe - "overcloker","MSIAfterburner"
9. Температура перегрева ВК - "tempalarm","75"
10. Минимальная температура ВК при которой произойдет перезагрузка - "tempRestart","40"
11. Дополнительное время ожидания в секундах после старта ВочДога. В течении этого времени ВочДог не будет реагировать на отсустие нагрузки от майнера  - "waittime","90"
12. Разрешить отправлять сообщение при старте вочдога - "start_message","1"
13. Тежим тестирования (заблокирована перезагрузка и выключение) - "test_mode","0"
14. Разрешить делать скриншоты при сбое перед рестартом - "ScreenShot","1"
15. Время перезагрузки при потери интернета в сек (если 0 перезагрузка не работает) - "NoInternetTimeRestart","0"
16. Время работы на экране - со старта системы "DisplayWorkTimeSystem","1" или старта ВочДога "DisplayWorkTimeSystem","0"
17. Параметр "Proxy", который определяет работу программы с Телеграмм без прокси при "Proxy","0" или с прокси "Proxy","IP:PORT", где IP - ip проски-сервера, PORT - порт проски-сервера.
18. Логин для прокси-сервера - "ProxyUser","0"
19. Пароль для прокси-сервера - "ProxyPassword","0"
20. Параметр "PingServer","google.com" задаёт сервер для теста пинга, т.е. проверки наличия интернета.
21. Обновляемые сообщения теперь могут сворачиваться если температуры на риге в норме. При нажатии "обновить" они примут форму полного сообщения на 15 минут. Запрепить\разрешить это режим можно установив параметр "GroupChatMinMessage" в конфиге.
22. "PCName_Length" определяет длину имени компьютера для команд с Телеграмма. Имя группы минус 2 символа.
23. "Proxy6APIkey" ключ из личного кабинета https://proxy6.net. Приобрести персональный Proxy-сервер для ВочДога со скидкой 5% - https://proxy6.net/?r=61093
24. "Coin_Address" - адрес кошелька  для получения статистики. Для miningpoolhub.com в параметре "Coin_Address" нужно указывать API_KEY из личного кабинета.
25. "Pool_Name" - название (адрес) пула для получение статистики(nicehash.com, zcash.flypool.org, ethermine.org, etc.ethermine.org, bsod.pw, zpool.ca, miningpoolhub.com). Если поставить любой символ вместо 0, то будет приходить информация о курсах BTC\RUB и BTC\USD.

++. TelNet - https://youtu.be/eKJXUacJyi4

Дополнительная информация может также присутствовать в файле ChangeLog.txt

По чтению хешрейта:

Для корректной работы этой функции необходимы условия:

1. Для семейства CCminer:
- Имя файла майнера: Ccminer.exe или Ccminer-x64.exe или Ccminer-x86.exe
- В бат-файле должен присутствовать ключ: -b 127.0.0.1:4068

2. Для DSTM:
- Имя файла майнера: zm.exe
- В бат-файле должен присутствовать ключ: --telemetry=127.0.0.1:2222

3. Для EWBF:
- Имя файла майнера: miner.exe
- В бат-файле должен присутствовать ключ: --api 127.0.0.1:42000

4. Для HSRMINER:
- Имя файла майнера: hsrminer_neoscrypt.exe или hsrminer_neoscrypt_fork.exe или hsrminer_neoscrypt_fork_hp.exe
- В бат-файле должен присутствовать ключ: -b 127.0.0.1:4067

5.  Для Claymore:
- Имя файла майнера: EthDcrMiner64.exe
- В бат-файле должен присутствовать ключ: -mport 127.0.0.1:3333

6. Для bminer:
- Имя файла майнера: bminer.exe
- В бат-файле должен присутствовать ключ: -api 127.0.0.1:1880

7. Для корректной работы этой функции необходимы условия:
- Имя файла майнера: PhoenixMiner.exe
- В бат-файле должен присутствовать ключ: -mport 127.0.0.1:3333
